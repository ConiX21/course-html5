jQuery.// Aliassé par $

Liste des paramètres acceptés :

  • jQuery(html)
  • jQuery(element)
  • jQuery(fonction)
  • jQuery(expr,context)

jQuery(html)

Crée l'arbre DOM de la chaine de caractère html passée en paramètre a la volée.

Paramètres:

  • html (String): code HTML

Cette methode renvoie: Objet Jquery

Prenons l'exemple suivant:

Cet exemple ajoute un div contenant bonjour a l'élément body.

$("<div><p>Hello</p></div>").appendTo("body")

jQuery(element)

Pointe sur l'élément ou le tableau d'élements passé en paramètre.

Paramètres:

  • element (element | array(elements)): élement ou tableau d'elements

Cette methode renvoie: objet JQuery

Prenons l'exemple suivant:

Affecte le style de la balise body, le fond du document deviendra noir.

$(document.body).css("background-color","black");

Prenons l'exemple suivant:

Cet exemple cachera tous les élements du formulaire nommé "monFormulaire".

$(monFormulaire.elements).hide();

jQuery(fonction)

Exécute la fonction lorsque que le document HTML est chargé. Equivaut à $(document).ready() Cela permet de ne commencer les opérations sur les différents éléments de la page uniquement lorsque ceux-ci sont complètement chargés.

Paramètres:

  • fonction (fonction): fonction a éxécuter lorsque le DOM sera chargé.

Cette methode renvoie: Objet JQuery

Prenons l'exemple suivant:

$(function(){ //contenu exécuté lorsque le document sera chargé });

jQuery(expr,context)

Élément très intéressant, cette fonctionnalité permet de passer en paramètre un sélecteur CSS ou XPath permettant de pointer sur un ensemble d'éléments.

Paramètres:

  • expr (String): selecteur
  • context (Element|jQuery): objet sur lequel se basera la recherche (optionnel)

Cette methode renvoie: Objet JQuery

Prenons l'exemple suivant:

Recherche tous les élements p qui se trouvent a l'intérieur d'un div.

$("div > p")

Testons sur le script suivant:

<p>one</p> <div><p>two</p></div> <p>three</p>

On obtiendra :

[ <p>two</p> ]

Prenons l'exemple suivant:

Recherche tous les champs input de type radio qui se trouvent dans le premier formulaire du document html.

$("input:radio", document.forms[0])

Prenons l'exemple suivant:

Recherche tous les éléments de type div situés dans le document XML.

$("div", xml.responseXML)
